# Мессенджер на Vue 3

Простое SPA-приложение мессенджера

## Стек

- Vue 3 + TypeScript
- Pinia
- Vue Router
- SCSS

## Что умеет
- История сообщений (генерируется автоматически)
- Отправка сообщений (Зашлушка с таймаутом)
- Статусы онлайн/оффлайн
- Сохранение в localStorage
- Счетчик непрочитанных сообщений
- Адаптив (мобилка)

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
